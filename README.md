# BernyAux

A simple auxiliary library to interface with PySCF and pyberny

## Dependencies
* [PySCF](https://github.com/sunqm/pyscf)
* [pyberny](https://github.com/azag0/pyberny)

## Installation
Add `$(pwd)/..` to the environment variable `$PYTHONPATH`
